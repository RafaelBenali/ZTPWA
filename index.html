Зачем это нужно?

ZT Валидатор — это PWA (Progressive Web App).
После первого открытия все файлы сохраняются на устройстве. Приложение работает полностью без интернета, никакие данные не отправляются на сервер — всё обрабатывается локально в браузере.

• Мгновенный запуск без браузера и адресной строки
• Работа без интернета в любом месте
• Иконка на рабочем столе / домашнем экране
• Полная конфиденциальность — текст не покидает устройство

Как установить

Android (Chrome)

1. Откройте эту страницу в Chrome
2. Нажмите меню (⋮) в правом верхнем углу
3. Выберите «Установить приложение» или «Добавить на главный экран»
4. Подтвердите установку

iPhone / iPad (Safari)

1. Откройте эту страницу в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой внизу экрана)
3. Пролистайте вниз и выберите «На экран Домой»
4. Нажмите «Добавить»

Компьютер (Chrome / Edge)

1. Откройте эту страницу в Chrome или Edge
2. Найдите иконку установки в правой части адресной строки (монитор с плюсом)
3. Нажмите «Установить»

Firefox

Firefox не поддерживает установку PWA. Приложение всё равно работает оффлайн, но без отдельной иконки. Используйте Chrome, Edge или Safari для установки.
